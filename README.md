# ✈️ Air Cargo SQL Analysis Project

## 📌 Overview

This project focuses on analyzing air cargo operations using MySQL. The analysis includes revenue trends, customer behavior, route performance, and time-based insights.

## 🧰 Tools Used

* MySQL

## 🔍 Key Analysis

### 📊 Revenue Analysis

* Total Revenue Calculation
* Revenue by Aircraft
* Monthly Revenue Trends

### 👤 Customer Analysis

* Top Customers by Revenue
* Customer Segmentation (High / Medium / Low)
* Customer Ranking using Window Functions

### ✈️ Route Analysis

* Busiest Routes
* Revenue by Route

### 📈 Advanced SQL

* Window Functions (RANK, DENSE_RANK, ROW_NUMBER)
* LAG & LEAD for trend analysis
* Running Totals
* CTE for structured queries

## 💡 Key Insights

* High-value customers contribute significantly to revenue
* Certain routes generate more traffic but not always high revenue
