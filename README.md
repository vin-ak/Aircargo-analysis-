✈️ Air Cargo Data Analysis Project

📌 **Overview**
This project focuses on analyzing air cargo operations using MySQL and Python. The analysis includes revenue trends, customer behavior, route performance, and time-based insights.

🧰 **Tools Used**

* MySQL
* Python (Pandas, Seaborn, Matplotlib)

🧹 **Data Cleaning & Preprocessing (Python)**

* Handled missing values and removed duplicate records using Pandas
* Converted data types for consistency (dates, numeric fields)
* Cleaned and prepared raw data before loading into MySQL

🔍 **Key Analysis**

📊 **Revenue Analysis**

* Total Revenue Calculation
* Revenue by Aircraft
* Monthly Revenue Trends

👤 **Customer Analysis**

* Top Customers by Revenue
* Customer Segmentation (High / Medium / Low)
* Customer Ranking using Window Functions

✈️ **Route Analysis**

* Busiest Routes
* Revenue by Route

📈 **Advanced SQL**

* Window Functions (RANK, DENSE_RANK, ROW_NUMBER)
* LAG & LEAD for trend analysis
* Running Totals
* CTE for structured queries

📊 **Data Visualization (Python - Seaborn)**

* Box plots to analyze price distribution and detect outliers
* Bar charts for revenue comparison across categories
* Trend visualizations for monthly revenue analysis

💡 **Key Insights**

* High-value customers contribute significantly to overall revenue
* Certain routes generate higher traffic but not always higher revenue
* Pricing distribution varies significantly across different ticket classes
* Identified outliers in pricing indicating potential anomalies or discounts
